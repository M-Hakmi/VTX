# Secondary structure notes

## PDB format

### Data types

3 data types possible, here is the mapping from PDB type to SS type:

- HELIX -> HELIX

- SHEET -> STRAND

- TURN -> COIL

### HELIX idendification code

- 1 = left alpha
- 2 = left omega
- 3 = left pi
- 4 = left gamma
- 5 = 3-10
- 6 = right alpha
- 7 = right omega
- 8 = right gamma
